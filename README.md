# Cloud Fun Facts App ☁️

A serverless AWS project that delivers witty cloud computing facts using Lambda, DynamoDB, and Amazon Bedrock (Nova Pro).

## Features
- Random cloud fun facts from DynamoDB
- Witty rephrasing with Generative AI
- REST API via API Gateway
- Fully CORS-enabled frontend for web apps

## Tech Stack
- AWS Lambda
- API Gateway
- DynamoDB
- Amazon Bedrock (Nova Pro)
- HTML/CSS/JS frontend
